##  기능 목록

---
- [ ] 카테고리별 메뉴 정보 저장

- [ ] 점심 메뉴 추천 시작
- [ ] 코치 이름 입력 받기
  - [ ] 2글자 ~ 4글자 
  - [ ] 초키는 최소 2명, 최대 5명
- [ ] 각 코치별 못 먹는 메뉴 입력받기
  - [ ] 못먹는 메뉴는 최소 0개 ~ 2개 (, 구분)

### 카테고리 선정
- [ ] 월요일 ~ 금요일까지 무작위로 뽑기
- [ ] 각 코치가 먹을 메뉴 추천
- [ ] 만약 못먹는 음식 추천되었다면 다시 뽑기!
- 
### 메뉴 추천
  - [ ] 한주에 같은 메뉴 추천 불가
  - [ ] 같은 카테고리는 최대 2
